HTML5新特性，增强型表单

  • 表单元素<input type=”?”>的值：

    1. 在HTML4中原有的type值:

    text, password, checkbox,radio, submit, reset, button, file, image, hidden,emai,number;

    1. HTML5中新增的type值：

    url, tel, search, range, color, date, month, week,datetime-local,time;

    下面是HTML5新增值的例子:

  1. type = “url”:

    • 解释：

      用于输入 url 的控件。类似 text 输入，但有验证参数，在支持动态键盘的设备上有相应的键盘。

    • 效果：

  1. type = “tel”

  • 解释:

用于输入电话号码的控件。拥有动态键盘的设备上会显示电话数字键盘。

  • 效果:

  1. type = “search”

    • 解释:

    用于搜索字符串的单行文字区域。输入文本中的换行会被自动去除。在支持的浏览器中可能有一个删除按钮，用于清除整个区域。拥有动态键盘的设备上的回车图标会变成搜索图标。

    • 效果:

  1. type = “range”

    • 解释:

    此控件用于输入不需要精确的数字。控件是一个范围组件，默认值为正中间的值。同时使用min和 max来规定值的范围。

    • 效果:

  1. type = “color”

    • 解释:

    用于指定颜色的控件；在支持的浏览器中，激活时会打开取色器。

    • 效果:

  1. type = “date”

    • 解释:

    输入日期的控件（年、月、日，不包括时间）。在支持的浏览器激活时打开日期选择器或年月日的数字滚轮。

    • 效果:

  1. type = “month”

    • 解释:

    输入年和月的控件，没有时区。

    • 效果:

  1. type = “week”

    • 解释:

    用于输入以年和周数组成的日期，不带时区。

    • 效果:

  1. type = “datetime-local”

    • 解释:

    输入日期和时间的控件，不包括时区。在支持的浏览器激活时打开日期选择器或年月日的数字滚轮。

    • 效果:

  1. type = “time”

    • 解释:

    用于输入时间的控件，不包括时区。

    • 效果:

  • 表单元素的新增属性:

    属性: 描述: 例子:
    placeholder 占位字符 <input type=”tel” placeholder=”请输入电话”>
    autofocus 自动获取输入框焦点 <input type=”text” autofocus>
    multiple 一个input是可以有多个值 <input type=”email” multiple>
    form 用于把输入框放置到form外部，使得input和form分离 <form id=”temp”></form> <input \type=”text” form=”temp” >
    required 必填项，内容不能为空 <input type = “text” required>
    minlength 指定字符串最小长度 <input type=”text” minlength=”4” maxlength=”16”>
    maxlength 指定字符串最大长度 <input type=”text” minlength=”4” maxlength=”16”>
    max 指定数字的最大值 <input type=”text” min=”0” max=”100”>
    min 指定数字的最小值 <input type=”text” min=”0” max=”100”>
    pattern 添加正则表达式 <input type=”tel” pattern=”1[3578]\d{9}”/>

  • 新增表单元素:

    1. HTML4中的表单元素:

      input, button, select/option, textarea, label

    2. HTML5中新增表单元素:

      datalist, progress, meter, output

    HTML5中新增表单元素例子:

    1. <datalist>

      • 解释:

      datalist 为input提供输入列表，用户可以从中选取，也可以输入,datalist”中定义的Id值名称要与“input”表单中“list”定义的值相同。

      • 效果:

    1. <progress>

      • 解释:

      用来显示一项任务的完成进度.虽然规范中没有规定该元素具体如何显示,浏览器开发商可以自己决定,但通常情况下,该元素都显示为一个进度条形式.

      • 效果:

    1. <meter>

      • 解释:

      <meter>元素用来显示已知范围的标量值或者分数值。用于识别一个值所处的范围，高于high（红色），处于low~high（黄色），低于low（绿色），大体上就是表示随着进度条的不同进度，其颜色也会不同，可以改变其宽、高属性。(min不写默认为0)

      • 效果:

    1. <output>

      • 解释:

      <output> 标签表示计算或用户操作的结果。语义标签,没有任何外观样式。

      • 效果:
